| --- | --- | --- |
| 已浏览 / 已看 / 想看 | 支持三种常用状态管理 | ✅ |
| 详情页状态操作 | 在详情页直接标记作品状态 | ✅ |
| 列表页状态显示 | 在列表中显示作品状态 | ✅ |
| 状态优先级显示 | 根据规则展示更重要的状态 | ✅ |

## 💾 115 网盘集成

| 功能 | 说明 | 状态 |
| --- | --- | --- |
| 启用 115 离线下载 | 在 `设置 → 115网盘` 中启用 | ✅ |
| OpenList 手动获取 | 借助 OpenList 获取 `refresh_token` / `access_token` | ✅ |
| 扫码授权 | 支持 OpenList 扫码和自有应用扫码 | ✅ |
| token 状态验证 | 验证并显示 token 有效状态 | ✅ |

| 115任务页 | 管理任务、搜索、刷新、清空、分页查看 | ✅ |
| 默认目录 ID / 验证次数 / 最大失败数 | 控制离线任务行为 | ✅ |
| 测试搜索 / 115 日志 | 用于排查和校验 115 链路 | ✅ |
