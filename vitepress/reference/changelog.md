## 如何更新

1. 打开 GitHub Releases 页面
2. 下载最新的 `javdb-extension-v*.zip`
3. 解压到固定目录
4. 进入 `chrome://extensions/` 或 `edge://extensions/`
5. 重新加载已解压扩展

## 版本说明

- `version.json` 中维护扩展主版本号，例如 `1.19.1`
- Git 标签额外带有构建号，例如 `v1.19.1.5`
- 从仓库标签观察，版本格式通常为 `主版本.次版本.修订号.构建号`
