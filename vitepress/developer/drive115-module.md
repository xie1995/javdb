# 115网盘离线下载功能

## 功能概述

本模块为JavDB扩展添加了115网盘离线下载功能，支持：

- 磁链自动离线下载
- 下载结果验证
- 批量下载管理
- 详细日志记录
- 验证码自动处理

## 使用前准备

1. **登录115网盘**：在浏览器中登录 https://115.com
2. **启用功能**：在扩展设置中开启115网盘功能
3. **配置下载目录**：设置默认下载目录（支持动态参数）

## 功能说明

### 1. 基础设置

- **启用115网盘功能**：总开关，控制是否启用115相关功能
- **下载目录**：指定离线下载的目标目录，支持以下动态参数：
  - `${#star}`：演员名称
  - `${#series}`：系列名称
  - `${#studio}`：制作商名称
  - 示例：`${#star}/云下载` 会创建以演员名命名的文件夹

### 2. 下载设置

- **验证次数**：下载后验证文件的重试次数（建议3-5次）
- **最大失败数**：批量下载时允许的最大失败次数
- **自动通知**：下载完成后是否显示通知

### 3. 使用方法

#### 单个下载
1. 在JavDB视频详情页，找到磁链区域
2. 点击磁链旁边的"离线下载"按钮
3. 等待下载完成和验证

#### 批量下载
1. 在有多个磁链的详情页
2. 点击"批量下载全部"按钮
3. 系统会依次下载所有磁链

#### 列表页快速访问
1. 在视频列表页，每个视频项会显示"115"按钮
2. 点击后跳转到详情页进行下载

### 4. 验证码处理

当115需要验证时：
1. 系统会自动弹出验证码窗口
2. 完成验证后窗口会自动关闭
3. 下载任务会继续执行

### 5. 日志查看

在扩展设置的"115网盘"标签页中：
- 查看下载历史和状态
- 导出日志文件
- 清空历史记录

在主日志页面中：
- 选择"115网盘"来源筛选115相关日志
- 查看详细的操作记录

## 技术架构

### 核心模块

- `api.ts`：115网盘API接口封装
- `verification.ts`：验证码处理
- `logger.ts`：日志管理
- `utils.ts`：工具函数
- `config.ts`：配置管理
- `types.ts`：类型定义

### API接口

- `getSign()`：获取离线下载签名
- `addOfflineTask()`：添加离线下载任务
- `searchFiles()`：搜索网盘文件
- `verifyDownload()`：验证下载结果

### 权限要求

扩展需要以下权限：
- `notifications`：显示下载通知
- `115.com`：访问115网盘
- `webapi.115.com`：调用115 API
- `captchaapi.115.com`：处理验证码

## 故障排除

### 常见问题

1. **下载失败**
   - 检查是否已登录115网盘
   - 确认网盘空间是否充足
   - 验证磁链是否有效

2. **验证码问题**
   - 确保浏览器允许弹窗
   - 手动完成验证码验证
   - 检查网络连接

3. **搜索无结果**
   - 确认115账号已登录
   - 检查文件排序设置（需要按修改时间排序）
   - 等待一段时间后重试

### 调试方法

1. **开启控制台**：按F12打开开发者工具
2. **查看日志**：在Console中查看详细错误信息
3. **测试功能**：在Console中运行 `testDrive115.all()` 进行功能测试

## 注意事项

1. **账号安全**：请确保115账号安全，不要在公共设备上使用
2. **网络环境**：建议在稳定的网络环境下使用
3. **存储空间**：注意115网盘的存储空间限制
4. **使用频率**：避免过于频繁的操作，以免触发限制
