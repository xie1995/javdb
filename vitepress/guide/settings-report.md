# 报告设置

报告设置对应仪表盘中的 `设置 → 报告`。

这一页控制的是：**报告统计口径和聚合参数**，不是替代报告页本身。

## 当前页真实能力

### 数据源与范围

当前设置项明确包括：

- 数据源模式
- 统计状态范围

这决定报告是更偏“观看记录”，还是会把浏览、想看等状态也纳入统计。

### 聚合参数

当前页还会影响：

- Top N 排行数量
- 其他聚合方式
- 本地统计整理方式
- 与 AI 提示词输入相关的参数

## 什么时候该先来这页

如果你发现：

- 报告排行不符合预期
- 想只统计“已看”或连“已浏览”一起算
- 想改变报告展示的数据口径

就应该先来这里，而不是在报告页反复生成同一套结果。

## 这页和报告页的关系

- `设置 → 报告`：改参数
- `报告`：看结果、导出、看历史

两者必须一起理解。
