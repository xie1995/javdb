# 磁力搜索

磁力搜索主要对应仪表盘中的 `设置 → 搜索引擎`，而实际使用通常发生在影片详情页。

你可以把它理解成：

- 设置页负责配置搜索源和策略
- 详情页负责展示和使用搜索结果

## 当前最重要的两部分

## 搜索引擎配置

在 `设置 → 搜索引擎` 中，你主要会处理：

- 启用或禁用搜索源
- 调整搜索策略
- 自定义搜索引擎配置
- 影响详情页搜索结果的来源和表现

## 详情页搜索使用

在影片详情页中，磁力搜索通常会和页面增强一起生效。

你实际会做的事情一般是：

1. 打开详情页
2. 查看自动搜索或手动触发后的磁力结果
3. 选择合适的资源
4. 按需复制、使用或 `推送115`

## 适合什么场景

### 想快速补全详情页资源来源

如果原页面资源不够理想，磁力搜索可以作为补充入口。

### 想把搜索和 115 连起来

比较常见的真实工作流是：

1. 在详情页看磁力搜索结果
2. 选一条更合适的链接
3. 执行 `推送115`

### 想调整搜索结果质量

这时应该去 `设置 → 搜索引擎`，而不是只在详情页里反复刷新结果。

## 当前更稳妥的理解方式

不建议把它理解成“一个单独页面里的搜索系统”，而更应该理解成：

- 它是详情页增强流程的一部分
- 它受设置页里搜索引擎配置影响
- 它经常和 115、标记、翻译一起使用

## 搜索失败时怎么排查

如果磁力搜索结果异常，优先检查：

1. 搜索引擎是否已启用
2. 页面增强相关功能是否正常工作
3. 网络是否影响部分搜索源
4. 是否只是当前作品没有理想资源
