# 全局操作

全局操作对应仪表盘中的 `设置 → 全局操作`。

顾名思义，这页负责的是：**不属于单一功能页、但会影响整体数据或整体状态的操作。**

## 典型用途

这类页面通常会承接：

- 导入数据
- 导出数据
- 清理数据
- 触发某些全局性的管理动作

## 什么时候来这里

如果你的目标不是“改某个功能的细设置”，而是：

- 整体导出一份数据
- 恢复某份导入数据
- 做一次大范围整理或清理

那就应该优先来这里。

## 使用建议

执行全局操作前，建议至少满足下面两点中的一点：

- 你已经清楚操作影响范围
- 你已经提前做过备份
