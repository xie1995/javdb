# 检查更新

检查更新对应仪表盘中的 `设置 → 检查更新`。

这页主要负责：**查看当前版本、检查最新版本、控制自动检查策略。**

## 当前页真实能力

根据当前界面，这里会处理：

- 当前版本
- 最新版本
- 上次检查时间
- 是否启用自动检查
- 检查间隔
- 是否包含预发布版本
- 手动立即检查
- 查看更新日志
- 下载更新

## 适合什么时候来这页

- 你想知道自己是不是最新版本
- 你不想手动频繁去 GitHub 看发布
- 你想决定是否接收预发布版本

## 自动检查怎么理解

自动检查的目标是“提醒你有新版本”，不是在后台自动完成安装。

因为当前扩展安装方式本身就偏手动，所以更新管理仍然是“发现新版本 + 跳转下载/查看日志”的思路。

## 推荐用法

- 稳定使用者：关闭预发布版本，只保留正常检查
- 深度测试者：可以打开预发布版本，尽早看到变化
