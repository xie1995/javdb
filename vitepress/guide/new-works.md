# 新作品监控

新作品监控对应仪表盘中的 `新作品` 标签页，主要用于追踪订阅演员后续的新发布内容。

它更像“发现入口”，而不是一页完成所有后续处理的终点页。

## 当前页更适合做什么

在当前版本里，新作品页最适合承担这些任务：

- 发现订阅演员是否有新作品
- 快速筛选感兴趣的内容
- 从新作品进入详情页继续处理
- 配合标记、磁力搜索和 `推送115` 使用

## 它和演员库的关系

这页的前提通常是你已经在 [演员管理](/guide/actor-management) 里做过订阅。

简单理解：

- `演员库` 负责管人
- `新作品` 负责看这些人有没有新内容

如果演员没有被订阅，这一页通常就不会是你的主工作区。

## 当前不建议怎样理解它

不建议把新作品页理解成：

- 自动完成所有下载流程
- 一键批量完成所有后续处理
- 不进详情页也能替代完整的单作品操作

更稳妥的理解是：

- 它负责“发现”
- 后续“标记 / 搜索 / 推送 / 翻译”仍然更适合在详情页完成

## 推荐工作流

### 工作流 1：演员订阅 → 新作品发现

1. 在演员库中订阅感兴趣的演员
2. 定期查看 `新作品`
3. 从列表中打开感兴趣作品
4. 进入详情页继续标记、搜索或推送115

### 工作流 2：新作品 → 想看 → 后续处理

1. 在新作品页先筛选出感兴趣内容
2. 先标成“想看”或加入自己的整理流程
3. 有时间时再回到详情页做进一步处理

### 工作流 3：新作品 → 详情页 → 115

1. 在新作品页发现目标作品
2. 进入详情页
3. 执行磁力搜索
4. 按需 `推送115`

## 当前与 115 的关系

当前版本不建议把“新作品页批量推送到 115”当成主流程。

更推荐：

- 在新作品页发现内容
- 再进入详情页逐条处理
- 或前往 `115任务` 页面手动补充任务

## 当前与标记的关系

新作品页更像一个“前置筛选层”。

后续你常会把它与这些操作连起来：

- 标记“想看”
- 标记“已看”
- 加入清单
- 详情页进一步整理
