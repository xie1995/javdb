# 快速开始

这份快速开始基于当前版本的真实界面编写，适合第一次安装后尽快把扩展跑起来。

## 你将完成什么

按照这份文档操作后，你会完成：

- 安装扩展并确认它已正常加载
- 打开弹窗和仪表盘
- 了解最常用的几个入口
- 完成最基础的同步、115、设置认知

## 安装扩展

### 方式一：下载预编译版本

1. 打开 [GitHub Releases](https://github.com/Adsryen/JavdBviewed/releases)
2. 下载最新的 `javdb-extension-v*.zip`
3. 解压到本地固定目录
4. 打开浏览器扩展管理页
   - Chrome：`chrome://extensions/`
   - Edge：`edge://extensions/`
5. 开启“开发者模式”
6. 点击“加载已解压的扩展程序”
7. 选择刚才解压后的目录

## 安装后先做这两件事

### 1. 打开扩展弹窗

点击浏览器工具栏中的扩展图标，你会看到当前版本的快捷弹窗。

弹窗里通常可以直接完成：

- 番号过滤相关开关
- 演员过滤相关开关
- 预览视频音量调整
- 列表显示控制
- 进入仪表盘/高级设置

如果弹窗能正常打开，说明扩展至少已经成功加载。

### 2. 打开仪表盘

从弹窗进入仪表盘后，你会看到当前真实的顶层标签：

- 首页
- 番号库
- 清单
- 演员库
- 新作品
- 数据同步
- 115任务
- 报告
- 设置
- 日志

这也是当前文档组织的主线。

## 第一次使用推荐顺序

### 第一步：先看“设置”

进入 `设置` 后，当前版本会看到这些设置卡片：

- 显示设置
- 功能增强
- 搜索引擎
- AI设置
- 隐私保护
- WebDAV同步
- 同步设置
- 115网盘
- Emby增强
- 报告
- 日志设置
- 高级配置
- 网络配置
- 全局操作
- 检查更新

建议第一次先看这几个：

1. [显示设置](/guide/settings-display)
2. [功能增强](/guide/page-enhancement)
3. [WebDAV 同步](/guide/webdav-sync)
4. [115 网盘集成](/guide/115-drive)

### 第二步：确认最基础的标记功能

访问 JavDB 详情页后，最先建议确认这几件事：

- 能否看到详情页顶部的状态操作区
- 能否把作品标记为“已看”或“想看”
- 返回列表页后，状态是否正确显示

如果这些都正常，说明最核心的本地功能已经可用。

### 第三步：按需要配置扩展能力

- 如果你重视备份和多设备一致性：先看 [WebDAV 同步](/guide/webdav-sync)
- 如果你需要把磁力任务送去 115：先看 [115 网盘集成](/guide/115-drive)
- 如果你更关注页面体验：先看 [页面增强](/guide/page-enhancement)
- 如果你希望统一整理记录：先看 [数据管理](/guide/data-management)

## 当前版本最常用的入口

### 弹窗

适合做即时开关和快速调整：

- 当前页面相关过滤
- 演员过滤
- 预览音量
- 列表列数/宽度

### 仪表盘

适合做集中管理：

- 番号库、清单、演员库、新作品
- 数据同步与 115任务
- 报告、设置、日志

### 详情页

适合做单作品操作：

- 标记状态
- 磁力搜索
- `推送115`
- 演员增强与信息操作

## 如果你只想最快上手

按下面顺序走就够了：

1. 安装扩展
2. 打开弹窗，确认能正常显示
3. 打开仪表盘，进入 `设置`
4. 调整 [显示设置](/guide/settings-display) 和 [功能增强](/guide/page-enhancement)
5. 打开一个作品详情页，测试“已看 / 想看”
6. 如需云端能力，再配置 [WebDAV 同步](/guide/webdav-sync) 或 [115 网盘集成](/guide/115-drive)
