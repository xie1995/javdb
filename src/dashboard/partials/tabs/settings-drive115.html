115 网盘离线下载

配置 115 网盘离线下载、扫码授权与任务验证能力。

模式与接口

启用 115 离线下载

支持三种入口：借用 OpenList（ 开源项目 ） 手动获取、借用 OpenList 扫码、或使用自有 115 应用扫码授权。

推荐没有 115 开放平台应用的用户先用 OpenList 方案；切换这里只影响上方授权区，不影响下方 token 手动填写。

可自定义 v2 接口基础域名，默认 https://proapi.115.com，无需以斜杠结尾。

OpenList 手动获取

适合没有自有 115 开放平台应用的用户。先通过 OpenList 的相关工具拿到 refresh_token / access_token，再粘贴到下方“凭据与状态”区域。

获取地址 https://api.oplist.org/
1. 打开上方地址后，选择 115 网盘 (OAuth2)。
2. 按页面提示跳转并登录 115 账号。
3. 勾选 使用 OpenList 提供的参数，再获取 token。
4. 把得到的 refresh_token 和 access_token 粘贴到下方。
5. 点击“验证有效性”，确认当前 token 可用。

扫码授权（PKCE）

当前使用内置 OpenList APP ID 进行扫码授权，无需手动填写。

流程：输入 APP ID → 生成二维码 → 用 115 手机客户端扫码并确认 → 自动保存新 token。

点击“生成二维码”开始授权
未开始授权
  • PKCE 模式不需要 App Secret。
  • 二维码失效后重新生成即可。
  • refresh_token 会用于后续自动刷新 access_token。

下载设置

收藏时自动推送下载

开启后，在 JavDB 点击收藏影片时，将自动获取该影片的磁力链接并择优推送到 115 网盘离线下载。择优规则：文件数量最少优先，文件数量相同时选文件最大的。

可手动填写目录 ID，也可通过文件夹选择器快速选择。

下载后验证文件的重试次数，建议 3-5 次。
