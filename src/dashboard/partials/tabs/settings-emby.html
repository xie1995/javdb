Emby联动

连接 Emby/Jellyfin 媒体库服务器，在浏览 JavDB 时为已入库影片添加标记。

开启后可以在视频封面显示已入库标记，并在本地维护一个媒体库索引。

服务器配置

配置 Emby 或 Jellyfin 服务器的连接信息。

选择您正在使用的媒体服务器软件。

输入您的 Emby/Jellyfin 服务器访问地址，包含协议和端口号。

在服务器管理面板 → API 密钥中创建新的 API 密钥。

可选。填写媒体库名称可只扫描该库，大幅加快同步速度。点击"获取库列表"可查看服务器上的所有媒体库。

库同步

选择媒体库数据的同步方式，定期更新本地影片索引。

实时查询：浏览JavDB列表/详情时在线查询Emby；定时同步：后台拉取Emby数据到本地缓存。自动抓取封面：同步后从JavDB获取影片标题和封面。

定时同步的时间间隔，建议设置为 24 小时(1440 分钟)或更长。

手动从服务器拉取当前所有影片的番号索引，用于标记已入库状态。

显示设置

控制在 JavDB 页面上的入库标记显示位置。

在搜索结果、想看列表、已看列表等页面的封面右上角显示已入库标记。

在单片详情页的封面右上角显示已入库标记。

保存设置

将当前的配置保存到本地存储中。

将服务器地址、API 密钥和同步模式等配置持久化保存。
