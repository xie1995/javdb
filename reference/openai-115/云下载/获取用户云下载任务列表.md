### 基本信息
Path 域名 + /open/offline/get_task_list   


Method：GET   


接口描述：获取用户云下载任务列表  


### 请求参数
### head
| 参数名称 | 参数值 | 是否必须 | 示例 |
| --- | --- | --- | --- |
| Authorization | Bearer access_token | 是 | Bearer abcdefghijklmnopqrstuvwxyz |


### 请求参数
**Query**

| 参数名称 | 是否必须 | 示例 | 备注 |
| --- | --- | --- | --- |
| page | 否 | 1 | 获取列表第几，如不传默认第一页 |


####注意事项 无

| 名称 | 类型 | 备注 | 其他信息 |
| --- | --- | --- | --- |
| state | boolean | | |
| message | string | | |
| code | number | | |
| data | array | | |
| ├─ page | int | 当前第几页 | |
